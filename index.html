Welcome to the Attain Web Interface (AWI), an easy to use REST API enabling seamless access to smart building data

Follow these steps to start accessing your data. Wherever you see <...> you will need to input the required project specfic information

  • Task

  • 1. Contact the Attain team or the customer administrator and request your user account be setup, if not already done
  • 2. Authenticate your user and obtain access token here : https://<host>/api/auth/login , and periodically refresh the token here : https://<host>/api/auth/token , All subsequent requests will require Bearer token in request header
  • 3. Fetch site details. Each site is represented as PROJECT_ABCD, where ABCD is a unique project reference : https://<host>/api/customers?pageSize=100&page=0 , Save the project ID for the project you wish to access
  • 4. Fetch project device (MEP devices) details : https://<host>/api/customer/<project ID>/devices?pageSize=500&page=0 , Save the device ID for the device(s) you wish to access
  • 4. Fetch project asset (building/floors) details : https://<host>/api/customer/<project ID>/assets?pageSize=50&page=0 , Save the asset ID for the asset(s) you wish to access
